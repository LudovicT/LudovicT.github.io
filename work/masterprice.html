MasterPrice

Référentiel de prix d'instruments financier et la suite logicielle servant à administrer ce référentiel.

MasterPrice est composé de :

  • Une IHM : MasterPriceGUI, client léger(Web Application) développé en ASP.NET MVC par mes soins
  • De nombreux batches remplissants divers rôles:
    • Récupération de données dans les systèmes
    • Envoye de données dans les systèmes
    • Vérification des données contenu dans la base de donnée Sql Serveur
    • Reporting de l'état du référentiel
