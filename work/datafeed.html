DataFeed

Suite logicielle qui récupère des données financières provenant de fournisseurs extérieur (Bloomberg, Factset, ...) pour être utiliser dans des systèmes de l'entreprise.

DataFeed est composé de :

  • Un ensemble de Windows Services permettant la communication avec Bloomberg via SFTP
  • Des batches récupérants des données de providers de données extérieur (Bloomberg, FactSet, ...) 
  • Un cache custom de données sous Sql Server pour éviter les temps d'attentes liée aux récupération via les providers..
  • Un Windows Service permetant de mettre des données en cache à partir de requètes utilisateur.
