Base de données

Création et maintien de Base de données

Les bases de données sont au cœur de la persistance de l'information. Il existe une multitude de bases de données. J’ai eu l’occasion d’utiliser deux types de bases de données : Oracle et SQL Server.

 

SQL

Le SQL est un langage d’interprétation de base de données. Aujourd’hui, je l’utilise avec SQL Server et Oracle. Il s’agit alors de supersets appelés respectivement T-SQL et PL/SQL. Désormais, en plus des requêtes classiques, j’utilise les vues SQL, les procédures stockées et les fonctions en SQL. La responsabilité de la gestion des données est alors donnée intégralement au moteur de gestion de bases de données relationnelles, gage de sécurité et de performance.

Ce langage est simple et efficace, il peut cependant devenir complexe et peut, par conséquent, devenir inefficace dans certains cas d’utilisations où il faudra savoir optimiser ses requêtes et placer des index aux bons endroits.

 

SQL Server

SQL Server est un serveur de gestion de bases de données relationnelles qui fonctionne uniquement sur les systèmes Windows. Robuste et puissant, je l’utilise quotidiennement lors de mes développements. J'ai eu l'occasion lors de mon expérience à Candriam de participer à la migration de MasterPrice d'un environement complètement Oracle vers un environement complètement SQL Server.

J'ai ainsi pu participer activement à la conception, la réalisation et surtout au maintien de cette nouvelle base de données.

Oracle

Oracle est aussi un serveur de gestion de bases de données relationnelles. Tout aussi robuste et puissant, je l’utilise aussi régulièrement. J'ai eu l'occasion lors de mon expérience à Candriam de participer à la migration de MasterPrice d'un environement complètement Oracle vers un environement complètement SQL Server.

Lors de mon travail sur la refactorisation de l'IHM MasterFile j'ai aussi eu l'occasion de beaucoup travailler sur ce moteur.

Projets concernés

MasterFile

Référentiel d'instruments financiers et la suite logicielle servant à administrer ce référentiel.

MasterPrice

Référentiel de prix d'instruments financiers et la suite logicielle servant à administrer ce référentiel.

DataFeed

Suite logicielle qui récupère des données financières provenant de fournisseurs extérieures (Bloomberg, Factset, ...) pour être utiliser dans des systèmes de l'entreprise.
